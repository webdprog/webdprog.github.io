Supabase Setup
Step 01 — Create project
supabase.com → New project
Name it "Spendly" and set a strong database password.
Step 02 — Create expenses table
SQL Editor → Run this:
CREATE TABLE expenses (
id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
name TEXT NOT NULL,
amount DECIMAL(10,2) NOT NULL,
category TEXT NOT NULL,
date TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
Step 03 — Enable RLS & policy
Allow full access (or adjust)
ALTER TABLE expenses ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public access" ON expenses FOR ALL USING (true);
Step 04 — Get credentials
Project Settings → API
Copy your URL and `anon` public key.
